This dashboard has been developed using node.js , , electron.js , d3.js v4.0.0 and Chromium .
The stats data are being fetch by Python Flask HTTP Api from SQLite3 Database running on Raspberry Pi Version 4 Model B.
The stats data are being generated by below three modules:
1. Voice Detection Module (Powered by:) running on POST Iot Device (Raspberry Pi)
2. Face Detection Module (Powered by: TensorFlowLite, OpenCV) running on POST Iot Device (Raspberry Pi)
3. Posture Detection Module (Powered by:) running on POST Iot Device (Raspberry Pi)
